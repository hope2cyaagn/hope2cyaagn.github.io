常用Git命令

Posted on 2023-03-25

Software Construction-Lab Git备忘录

学习记录

  • git commit //提交

  • git branch <分支名> //创建分支

  • git checkout

    • git checkout <分支名> //切换到分支
    • git checkout <提交记录> //HEAD指向该提交记录
    • ^ //向上移动1个提交记录(父结点)
      ~<num> //向上移动num个提交记录

  • git merge <分支名> //把<分支名>合并到当前分支

  • git rebase <分支名> //复制副本并合并

  • git log //查看提交记录的哈希值

  • -f //分支指向另一个提交

    git branch -f main HEAD~3 //将main分支强制指向HEAD的第3级父提交

  • 撤销

    • git reset //向上移动分支git reset HEAD~1

    • git revert //撤销更改并分享给他人(远程)git revert HEAD

    • git cherry-pick //将<提交记录>复制到当前分支git cherry-pick C2 C4

  • rebase -i交互式rebase git rebase -i HEAD~4

    • 调整提交记录的顺序
    • 删除不想要的提交(切换pick状态)
    • 合并提交

  • git clone //远程拉取到本地
